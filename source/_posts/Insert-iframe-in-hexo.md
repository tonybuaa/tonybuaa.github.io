如果想要在文章中插入iframe，可以使用iframe blockquote：
```
{% iframe url [width] [height] %}
```

下面是一个例子：
```
{% iframe https://www.google.com/maps/d/embed?mid=1W1e-nBuh3wSXZCpCgS0m6RJnpgJDa8rH 640 480 %}
```
